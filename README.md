# DVWA-SQL-Cli
Short python CLi that will detect an sql injection vulnerability



######**Requirements:**
 	- [x] Python3
 	- [x] virtualenv

######**Setup:**
 	- cd into project
 	- python main.py  // to see menu
  
######**Usage:**
  To see menu:
    - python main.py 
      
  To Run Application:
  ```
  python main.py start <URL of DVWA>
  ```
